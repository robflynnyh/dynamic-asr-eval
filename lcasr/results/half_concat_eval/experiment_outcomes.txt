Half-concat experiment outcomes (earnings22 test, epoch 1, 3 repeats)
================================================================

Setup reminder
--------------
These runs use the half-concat within-dataset evaluation:
- split the 6 test recordings into two halves
- concatenate one half into a single adaptation spectrogram
- adapt on that concatenated half
- evaluate on the held-out half
- swap halves and average the two held-out WERs
- repeat 3 times

What this likely means
----------------------
The main signal so far is:
- lowering LR from 9e-5 to 9e-6 helped a lot
- at lr=9e-6, adapt_overlap=14336 outperformed adapt_overlap=0 in this half-concat setup

Current ordering (best to worst):
1. Run C: ao=14336, lr=9e-6  -> 0.1787727500673618
2. Run B: ao=0,     lr=9e-6  -> 0.18129123025301938
3. Baseline:                    0.183466601095398
4. Run A: ao=14336, lr=9e-5  -> 0.1890249398203365

So the strongest effect appears to be the LR drop, and among the low-LR settings tested so far, matched overlap is currently best.
